# 🚀 AURA ERP – Procurement & Vendor Management System

## 📌 Overview

AURA ERP is a modern Procurement & Vendor Management System designed to streamline the complete procurement lifecycle within an organization.

The platform enables procurement officers, vendors, managers, and administrators to collaborate efficiently through a centralized digital workflow, reducing manual processes and improving transparency.

---

## ✨ Key Features

### 🔐 Authentication & Role Management

* Secure Login & Signup
* Role-Based Access Control
* Forgot Password Functionality
* Multiple User Roles:

  * Officer
  * Vendor
  * Manager
  * Administrator

### 📊 Dashboard

* Active RFQ Monitoring
* Pending Approval Tracking
* Purchase Order Statistics
* Vendor Overview
* Recent Activity Timeline

### 🏢 Vendor Management

* Vendor Registration
* Vendor Directory
* Category-Based Filtering
* GST Information Management
* Vendor Ratings & Status Tracking

### 📝 Request for Quotation (RFQ)

* Create New RFQs
* Add Multiple Procurement Items
* Set Submission Deadlines
* Upload Attachments
* Assign Vendors

### 💰 Vendor Quotation Submission

* View Assigned RFQs
* Submit Pricing Details
* Delivery Timeline Entry
* Additional Remarks & Notes

### ⚖️ Quotation Comparison

* Compare Multiple Vendor Quotations

* Delivery Comparison
* Vendor Performance Evaluation
* Best Offer Identification

### ✅ Approval Workflow

### 📧 Internal Mailbox

* Audit Trail

* Procurement Spend Analysis
